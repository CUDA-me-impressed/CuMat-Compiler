# CuMat Compiler
University of Warwick 4th Year Group Project. 
Creating a compiled functional language for matrices optimised for CUDA multiprocessing.

# Requirements

1. Clang 10
2. llvm-10
3. llvm-10-dev
4. openjdk-8-jre 
5. pkg-config
6. uuid-dev
7. CUDA toolkit >=11.3
8. nvcc
9. CUDA Driver supporting language features >=50.1
